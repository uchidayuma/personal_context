# L2: Character & Talent（気質・才能）

> Zone: **CORE** — 変わらない自己
> 生まれ持った傾向。経験で変化しにくい気質レベルの特性。

---

## 基本的な気質

- 内向的 / 外向的: 内向的（少人数で深い話を好む。多人数での表面的な会話は消耗する）
- 直感的 / 分析的: 構造的・分析的（複雑なことをシンプルに分解するのが自然）
- 感情的 / 論理的: 体の感覚ベースで判断する（論理より感覚を最終判断軸にする）

## 自然にやってしまうこと

- 構造的に考える（周囲から褒められるが本人は当たり前と思っている）
- 複雑なことをシンプルに分解する
- 相手の表情・感情を読もうとする（傾聴・察する癖がある）
- 他人のメンタル面を自然と気にかける
- 戦略を考える（自分らしいと感じる瞬間のひとつ）

## 充実したときの動機の種類

- 没頭感（音楽・ポッドキャスト収録・ビジネスモデルを考えるとき時間が溶ける）
- 自由感（自分で決めて動けた、縛られなかった）
- 達成感（複雑な部分が解決できたとき、仕事がはかどって周りをフォローできたとき）

## 思考のクセ

- お金が切迫すると「とにかく稼がないと」モードになり、体の感覚が遠くなる → 長期判断が効かなくなる
- 体の感覚（「嫌だ」というサイン）を自分の判断基準として重視する傾向がある
- 周囲の意見より自分の意志を優先して進路を決める傾向がある

## 気分が上がる状況

- 自分のペースで動けている、自分の得意を活かしている
- 論理的な会話ができる相手といる
- オリジナリティが発揮されているとき
- 精神面・内観しているとき
- バックパック1個で動けるような身軽な状態

## 気分が下がる状況

- 長期間縛られる契約・管理下に置かれる働き方
- 自分の良いところを活かせない役割、悪い部分を否定され続ける環境
- 圧力をかけてくる人、声の大きい感情的な人
- 多人数での表面的な会話
- 細かい・正確性が命の業務

## 体のシグナル

- 体が軽くなる・肩や胸が広がる → 心地よい状態のサイン（その方向に進んでいい）
- 胃のあたりが重い感覚・肩が丸まって呼吸が浅くなる → 不快・危険な状態のサイン
- 声が出しづらくなる → 限界・無理をしているサイン

## コミュニケーションの特徴

- 傾聴・察するのが得意（内向的だからこそ相手の感情を読もうとする）
- 少人数での深い会話は得意、多人数での表面的な会話は消耗する
- 褒められることより率直なフィードバックを好む
