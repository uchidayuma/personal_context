# Output Examples

Sample exported Markdown files generated by Personal Context Engine.

- `ja/` — Japanese session output
- `en/` — English session output

## Generate your own

```bash
pnpm tsx scripts/export-example.ts ja    # export to output_example/ja/
pnpm tsx scripts/export-example.ts en    # export to output_example/en/
```

The script reads from your local SQLite database (`DB_PATH` in `.env`) and writes one Markdown file per context layer.
